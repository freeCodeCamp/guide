## Create a Simple JSX Element

This challenge is fairly simple, replace the div in the JSX element with h1
  ```javascript
  const JSX=<h1></h1>;
  ```
Then add the required text.
  ```javascript
  const JSX = <h1>Hello JSX!</h1>;
  ```
