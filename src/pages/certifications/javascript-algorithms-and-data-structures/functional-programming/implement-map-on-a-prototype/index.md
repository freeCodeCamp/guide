## ![:beginner:](https://forum.freecodecamp.com/images/emoji/emoji_one/beginner.png?v=3 ":beginner:") Basic Code Solution:
```javascript
    Array.prototype.myMap = function(callback) {
      var newArray = [];
  
      s.forEach(each => {
        newArray.push(callback(each));
      })
  
      return newArray;
    };
```

### Code Explanation:

*   First, we declare a new empty array. We then loop over the elements in the `s` array using the forEach method, applying the callback to each item, and then pushing them in the newly declared array. Finally, we return the new array.
