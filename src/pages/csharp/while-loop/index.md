# While Loop

The while loop executes a block of code until a specified condition is false.

## Example
```
int i = 0;
while (i < 5)
{
    Console.WriteLine("Number " + i);
    i++;
}
```

## Output:
```
> Number 0
> Number 1
> Number 2
> Number 3
> Number 4
```
