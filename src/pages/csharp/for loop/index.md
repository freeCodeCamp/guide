<h1>for loops</h1>
A for loop will execute a block of statments inside of the loop until the statement returns false.

<h2>Examples</h2>
This loop will execute 10 times starting at 0 and ending at 9.

```
for(int i = 0; i<10; i++)
{
  Console.Writeline(i);
}
```

<h3>Output</h3>

```
1
2
3
4
5
6
7
8
9
```
