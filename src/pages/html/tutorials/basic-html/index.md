## Basic HTML

Hyper Text Markup Language (HTML) is the programming language used to build web pages.

Here is a basic web page created in HTML:
```html
<!DOCTYPE html>
<html>
  <head>
    <title>Basic HTML Page</title>
  </head>
  <body>
    <p>Hello World!</p>
  </body>
</html>
```
