## Image Galleries

CSS can be used to create an image gallery.

<b> Example </b>

<b> CSS </b>
```
div.gallery {
    margin: 5px;
    float: left;
    width: 180px;
    border: 1px solid #ccc;
}

div.gallery img {
    height: auto;
    width: 100%;
}

```

<b> HTML </b>
```
<div class="gallery">
    <img src="image_name.jpg">
</div>

<div class="gallery">
    <img src="image_name.jpg">
</div>

```
