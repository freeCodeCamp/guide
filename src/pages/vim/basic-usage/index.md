## VIM Basic Usage
  

### Open File
- Run vim or vi and open the given filename.

### Save File
- :w

### Save and Exit
- :x 
- SHIFT ZZ
- :wq

### Exit file if no changes made
- :q

### Exit file and undo any changes made
- :q!

### Display line numbers
- :set nu

## About Vim
