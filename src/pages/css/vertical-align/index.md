## Vertical Align

The vertical-align CSS property controls the vertical alignment of an inline or table-cell element.

#### Example
```css
img {
  vertical-align: 7em;
}
       
```

#### Possible values
* baseline
* sub
* super
* text-top
* text-bottom
* middle
* top
* bottom
* *length* (e.g. px, %, em, etc.)
* inherit
* initial
* unset
