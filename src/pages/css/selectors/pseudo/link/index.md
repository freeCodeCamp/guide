## Link

The :link CSS pseudo-class is used to target all unvisited anchor (<a>) elements that have an href attribute, even if the href has an empty value.

Syntax:

```css
:link
```

## Example

```css
/* Selects any <a> that has not been visited yet */
a:link { 
    color: red;
}
```
