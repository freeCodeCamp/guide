## Updating Object Properties
## Hint:
Use dot ** . ** notation to access the object property.

## Spoiler Alert: Solution Ahead!

## Solution:

```javascript
// Setup
var myDog = {
  "name": "Coder",
  "legs": 4,
  "tails": 1,
  "friends": ["freeCodeCamp Campers"]
};

// Only change code below this line.

myDog.name = "Happy Coder"; // Solution
```
