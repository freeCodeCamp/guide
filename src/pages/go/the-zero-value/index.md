## The Zero Value
Understanding some zero values inside Golang:
1. false for booleans
2. 0 for integers
3. 0.0 for floats
4. "" for strings
5. nil for 
   * pointers
   * functions
   * interfaces
   * slices
   * channels
   * maps

Please, use short declaration operator as much as possible.

* Use var for:
  * zero value
  * package scope
