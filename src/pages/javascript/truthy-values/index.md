---
title: Truthy Value
---
A **truthy** value is a value that translates to **true** when evaluated in a _Boolean_ context.

All values are **truthy** unless they are defined as **falsy** (i.e. except for `false`, `0`, `""`, `null`, `undefined` and `NaN`).
