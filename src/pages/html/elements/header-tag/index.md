## Header Tag

``<header>`` is one of the new semantic tags in HTML5.

It specifies that its content is introductory. 

``<header>`` is sometime used for navigation links.

### What to put in ``<header>`` element:

Logo

Name

### Examples

``` html

<header>
  <h1>Hello World</h1>
  <p>This is a Hello World</p>
</header>

```
