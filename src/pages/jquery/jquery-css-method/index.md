## jQuery CSS Method

We can change the CSS of an HTML element directly with jQuery.

jQuery has a function called `.css()` that allows you to change the CSS of an element.

    <script>
      $(document).ready(function() {
        $("#target1").css("color", "red");

      });
    </script>
