## Use jQuery to Modify the Entire Page

### Problem Explanation

Add the classes `animated` and `hinge` to your `body` element.

#### Relevant Links:
  - [.addClass()](https://api.jquery.com/addClass/e)

### Solution:
```javascript
<script>
  $("body").addClass("animated hinge");
  });
</script>
```
