The HTML <tr> tag is used for defining a table row.

Example :

<html>

   <head>
      <title>HTML tr Tag</title>
   </head>

   <body>
      <table border = "1">
         <tr>
            <th>Football Players</th>
            <th>Ranking</th>
         </tr>
         <tr>
           <td> Cristiano Ronaldo </td>
           <td>1</td>
        </tr>
         <tr>
            <td>Lionel Messi </td>
            <td>2</td>
         </tr>
         
      </table>
   </body>

</html>
