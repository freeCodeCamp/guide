## Text

CSS styles next in a number of ways including the following:

* Color 
* Alignment 
* Decoration (underline, strikethrough, etc.)
* Transformation (uppercase, lowercase, etc.) 
* Indentation 
* Line spacing 
* Line height
* Direction
* Word Spacing
* Text Shadow
