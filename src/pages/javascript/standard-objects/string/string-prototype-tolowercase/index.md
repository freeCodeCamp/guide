## String.prototype.toLowerCase

The JavaScript method **```.toLowerCase()```** returns the string it was called on, converted to lower case.
##### Syntax
```
str.toLowerCase();
```

#### Examples:
``` console.log("ELEPHANT".toLowerCase());  // the console will log "elephant" ```
