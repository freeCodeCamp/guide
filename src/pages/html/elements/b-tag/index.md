## B Tag

The text inside `<b>` tag will be in **bold** style.

#### Example:
```HTML
<!DOCTYPE html>
<html>
  ...
  <b>I'm inside b tag</b>
  I'm outside b tag
  ...
</html>
```

#### Result:
<pre>
<b>I'm inside b tag</b>
I'm outside b tag
</pre>
