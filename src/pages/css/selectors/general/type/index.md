## Type

<p>Type Selector selects elements by node(tag) name.</p>

####Code Syntax</h4>

```css
* { 
    css declarations; 
}
```

####Code Example</h4>
```css
h1 { 
    background-color: red;
}
```
selects all h1 element & set background color red.

```css
div { 
    width: 100px;
}
```
selects all div element & set element width 100px.
