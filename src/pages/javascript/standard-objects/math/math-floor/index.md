## Math Floor

`Math.floor(x)`
function returns the largest integer less than or equal to a given number.

#### Example
```javascript
Math.floor( 45.95); //  45
Math.floor( 45.05); //  45
Math.floor(  4   ); //   4
Math.floor(-45.05); // -46 
Math.floor(-45.95); // -46
```
