## Font Color Attribute

This attribute specifies the color of the text inside a
  `<font>
  `
element.

Syntax: 

`<font color="red">
`

Example:
```html
<html>
  <body>
    <font color="red">This is awesome!</font>
  </body>
</html>
```

Note : `The color attribute of <font> is not supported in HTML5. Use CSS instead.`
