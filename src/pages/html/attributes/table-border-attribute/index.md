## Table Border Attribute

The border attribute specifies if a border should be displayed around the table cells or not.
The following HTML table will be displayed with borders around the table cells:
<table border="1">
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$100</td>
  </tr>
  <tr>
    <td>February</td>
    <td>$80</td>
  </tr>
</table>
Use css because the border attribute of <table> is not supported in HTML5.
