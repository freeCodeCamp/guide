##Jupyter Installation:


1. Install Python:
    ```
    $ [sudo] apt-get install python
    ```
    or
    ```
    $ [sudo] apt-get install python3
    ```
    
2. Install pip:
    ```
    $ sudo apt-get install python-pip
    ```
3. Install Jupyter
  ```
  $ sudo pip3 install jupyter
  ```
4. If you encounter an error try:
  ```
  $ pip3 install --upgrade pip
  ```
