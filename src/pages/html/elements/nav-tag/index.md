## Nav Tag

The purpose of the `<nav>` element is to provide a section of a webpage that contains links to other sections in the same document or to other documents.

### Example

```html
<html>
  <body>
    <nav class="menu">
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
  </body>
</html>
```
