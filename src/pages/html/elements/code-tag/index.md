## Code Tag

It defines a piece of code. It gives users a feel of code upon looking the text entered in between the `<code>` tags. We can style it in CSS as needed.
<!-- The article goes here, in GitHub-flavored Markdown. Feel free to add YouTube videos, images, and CodePen/JSBin embeds  -->

```html
<html>
    <head>
        <title>Document Titles Belong in the Head</title>
    </head>
    <body>
        <p>This is normal text!</p>
        <code>This looks like a code</code>
    </body>
</html>
```
