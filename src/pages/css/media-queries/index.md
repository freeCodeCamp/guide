### What are media queries?
Starting with HTML 4 and CSS2 there is a way of creating custom stylesheets depeding on the device propreties that uses the website. For example, if you want to have a sidebar on larger screens but want to hide it on smaller ones you can do it as follows:

Let's say our sidebar is inside of a div

```html
<div id="sidebar"> ...
</div>
```
