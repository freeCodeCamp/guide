# For Loop

A for loop is a repetition control structure that executes a loop a specific number of times.

## Example
```
for (int i = 0; i < 5; i++)
    {
      Console.WriteLine("Number {0}", i);
    }
```

## Output:
```
> Number 0
> Number 1
> Number 2
> Number 3
> Number 4
```
