## Elements

The smallest building blocks of React apps are elements which simply describe what you want to see on the screen. 

```js
const element = <p>I am an Element!</p>;
```

Elements are the building blocks of components. 

For more information head to <a href='https://reactjs.org/docs/rendering-elements.html' target='_blank' rel='nofollow'>https://reactjs.org/docs/rendering-elements.html</a>
