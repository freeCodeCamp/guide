## ![:beginner:](https://forum.freecodecamp.com/images/emoji/emoji_one/beginner.png?v=3 ":beginner:") Code Solution:

```javascript
function isLess(a, b) {
  // Fix this code
  return a < b;
}
// Change these values to test
isLess(10, 15);
```
## Code Explanation:
We can return the comparison directly from the function without the use of if-else statements to make code more concise and neat.
