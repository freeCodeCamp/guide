## Create Your First CSS Grid

### Hint 

To make any div a grid we use the following syntax.
  ````css
  display:grid;
  ````

### Solution

We simply change the style of the .container to
````css
  .container {
    font-size: 40px;
    width: 100%;
    background: LightGray;
    display: grid;
  }
  ````
