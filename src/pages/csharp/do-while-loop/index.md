# Do While Loop

The do while loop executes a block of code once and then it keep executing it until a specified condition is false.

### Example
```csharp
int i = 0;
do
{
    Console.WriteLine("Number " + i);
    i++;
}
while (i < 5);
```

### Output:
```
> Number 0
> Number 1
> Number 2
> Number 3
> Number 4
```
