## Tr Tag

The `<tr>` tag defines a standard row in an HTML table. You can then use a mix of <td> (data cell) and <th> (header cell) elements in a row.

### Example
```html
<html>
    <head>
      <title>Tr Tag Example</title>
    </head>
  <body>
    <table>
      <tr>
        <th>Header1</th>
        <th>Header2</th>
      </tr>
      <tr>
        <td>Cell A</td>
        <td>Cell B</td>
      </tr>
    </table>
  </body>
</html>
```
