## Basic HTML

when you start project with html. You must create html declaration. 
this is example.

''''<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html> ''''

this is minimum to create project html.
