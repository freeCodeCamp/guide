## Array.prototype.splice

The  `splice()` method removes and/or adds elements within an array.

## Example
``` javascript
var myArray = ['a', 'b', 'c', 'd'];
letters.splice(4, 0, 'e'); // insert 'e' at 2-index
// myArray is now ["a", "b", "c", "d", "e"]
```
