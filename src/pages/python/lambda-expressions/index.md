## Lambda Expressions

In Python, lambda expressions are anonymous functions (functions with no names). Instead of using `def` to define a function, we can instead use `lambda`.
<br><b>Ex:<b>
```
def add1(x,y):
    return x + y 
    
add2 = lambda x, y: x + y

print(add1(3, 2)) # prints 5
print(add2(3, 2)) # prints 5
```
