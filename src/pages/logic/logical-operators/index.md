## Logical Operators

* **And** (&&)
* **Or** (||)
* **Not** (!)
* **If**
* **Iff** (if and only if)
* **Xor** ("eXclusive or")

#### Examples:
And:
```
print(True and False)
// False
print(True and True)
//True
```

Or: 
```
print(True or False)
//True
print(False or False)
//False
```

Not:
```
print(not True)
//False
print(not False)
//True
```
