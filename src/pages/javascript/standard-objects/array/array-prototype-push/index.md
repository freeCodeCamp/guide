## Array.prototype.push

The 'push' method adds one or more elements to the end of an array.

```javascript
var colors = ['Red', 'Green', 'Purple'];
colors.push('Yellow', 'Pink');

console.log(colors);
// results in ['Red', 'Green', 'Purple', 'Yellow', 'Pink'];
```
