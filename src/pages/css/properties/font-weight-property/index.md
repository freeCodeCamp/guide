## Font Weight Property

The `font-weight`property specifies the weight (or boldness) of the font. Accepts keywords (`bold`, `normal`, `bolder`, `lighter`) or numeric keywords (`100`, `200`, `300`, `400` etc.) `400` is the same as `normal`.

```css
.bold {
   font-weight: bold
}

.light-weight {
   font-weight: 100
}
```
